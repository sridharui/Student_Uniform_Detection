================================================================================
                        TROUBLESHOOTING
================================================================================

❌ "Cannot connect to 192.168.0.9"
   → Check phone IP is correct
   → Ensure phone is on same WiFi network
   → Ping 192.168.0.9 from computer

❌ "SSL Certificate Error"
   → Use --http flag: python mobile_webcam_detector.py --http

❌ "No detections"
   → Improve lighting (natural light recommended)
   → Stand 1-3 meters from camera
   → Wear complete uniform
   → Try desktop webcam test first

❌ "Low FPS / Slow detection"
   → Increase skip frames: --skip 3 or --skip 4
   → Reduces accuracy slightly but speeds up processing
