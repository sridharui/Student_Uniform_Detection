  OPTION 1: Interactive Menu (Recommended)
  $ python quick_start_boys_girls.py

  OPTION 2: Direct Webcam Detection
  $ python uniform_detector_system.py --no-serial

  OPTION 3: Using Batch Script (Windows)
  $ run_uniform.bat

  OPTION 4: Using PowerShell (Windows)
  $ .\run_uniform.ps1

WHAT'S IN THE MAIN SYSTEM
================================================================================

✅ AUTOMATIC GENDER DETECTION
   - Detects BOYS (Shirt) vs GIRLS (Top)
   - Confidence-based classification

✅ GENDER-SPECIFIC REQUIREMENTS
   - BOYS: ID Card + Shirt + Pant + Shoes
   - GIRLS: ID Card + Top + Pant + Shoes

✅ CLASS-SPECIFIC CONFIDENCE THRESHOLDS
   - Shirt (boys): 0.55
   - Top (girls): 0.55
   - Other items: 0.50

✅ REAL-TIME DETECTION
   - Webcam support
   - Frame-by-frame analysis
   - Serial output (optional Arduino)

✅ DETAILED OUTPUT
   - Gender classification
   - Detected items list
   - Missing items list
   - Confidence scores
   - Status flag (1=complete, 0=incomplete)
